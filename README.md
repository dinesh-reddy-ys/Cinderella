# AutomationProject

## 🚀 Features
- Page Object Model
- Jenkins integration
- Docker compatibility
- Multi-browser testing

## 🛠️ Tech Stack
- Java
- Selenium
- TestNG
- Maven
- GitHub Actions / Jenkins
